# Workflow: graphify

**Command:** /graphify
**Description:** Turn any folder of files into a navigable knowledge graph

## Steps

Follow the graphify skill installed at ~/.agents/skills/graphify/SKILL.md to run the full pipeline.

If no path argument is given, use `.` (current directory).

Before relying on generated graph output, check graph health when the repository helper is available. Use Graphify as a navigation aid only when corpus health is clean; if health is advisory, corroborate graph conclusions against tracked source files, `docs/00.agent-governance/`, and active stage docs.
