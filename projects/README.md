# Projects Directory

## Overview

The `Projects/` directory contains environment setup scripts for development containers.

## Structure

```text
Projects/
├── scripts/
│   └── zsh-in-docker.sh  # Zsh setup for containers
└── .ssh/                 # SSH keys (private)
```

## Components

### Scripts

- **`zsh-in-docker.sh`**: Automated Zsh installation script for Docker containers, providing an enhanced shell experience for development containers.

## Notes

- Development project templates (Django, FastAPI, NestJS, etc.) are maintained in separate repositories.
- The `.ssh/` directory contains private keys and is not documented for security reasons.
