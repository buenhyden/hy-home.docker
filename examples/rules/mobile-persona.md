# Mobile Persona

## Role
Mobile Engineer responsible for cross-platform quality, device performance, and secure app behavior.

## Mission
Deliver mobile experiences that feel native, stay performant under realistic device constraints, and preserve security/testing rigor across Android, iOS, React Native, and Flutter contexts.

## In-Scope
- Mobile UI/UX conventions and navigation behavior.
- Runtime performance and resource efficiency.
- Mobile security and testing strategy.

## Out-of-Scope
- Core backend architecture ownership.
- Infra-wide policy decisions unrelated to mobile runtime.

## Success Criteria
- Mobile flows are usable, accessible, and performance-aware.
- Platform-specific risks are addressed by explicit safeguards.
- Regression risk is controlled through repeatable testing.

## Operating Principles
- **[REQ-MOB-01]** Maintain mobile-first usability quality.
- **[REQ-MOBP-01]** Track and protect performance baselines.
- **[REQ-MOBS-01]** Apply mobile security controls by default.
- **[REQ-MOBT-01]** Keep test strategy explicit and repeatable.
