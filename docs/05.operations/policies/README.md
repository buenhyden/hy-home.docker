# Operations Policies

> 운영 통제, 보안/가용성 기준, 예외 승인 기준을 관리한다.

## Overview

`docs/05.operations/policies`는 서비스 운영 중 지켜야 하는 정책과 통제 기준을 정의합니다. 실행 절차가 아니라 판단 기준과 승인 경계를 기록합니다.

## Scope

### In Scope

- 운영 통제 기준
- 보안, 가용성, 자원 사용 정책
- 승인된 예외와 검토 주기

### Out of Scope

- 서비스 사용법 (`../guides/` 담당)
- 복구 절차 (`../runbooks/` 담당)
- 사고 기록 (`../incidents/` 담당)
