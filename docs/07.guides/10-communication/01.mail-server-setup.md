# 📧 Mail Server Setup (Stalwart)

이 가이드는 `hy-home.docker` 환경에서 Stalwart 메일 서버를 설정하고 도메인을 통합하는 방법을 설명합니다.

## 1. 개요
Stalwart는 SMTP, IMAP, JMAP을 지원하는 차세대 올인원 메일 서버입니다. Rust로 작성되어 성능이 뛰어나며 통합된 보안 기능을 제공합니다.

## 2. 사전 준비 사항
- **포트 개방**: 호스트 시스템에서 25(SMTP), 587/465(Submission), 993(IMAPS) 포트가 사용 가능해야 합니다.
- **인증서**: `secrets/certs` 내에 유효한 도메인 인증서가 있어야 합니다.
- **비밀번호**: Docker Secret `stalwart_password`를 미리 생성해야 합니다.

## 3. 설정 및 배포
1. `infra/10-communication/mail` 디렉토리로 이동합니다.
2. 서비스를 시작합니다:
   ```bash
   docker-compose --profile communication up -d stalwart
   ```
3. 관리자 UI(`https://mail.${DEFAULT_URL}`)에 접속하여 로그인을 확인합니다.

## 4. 도메인 연동 (SPF/DKIM/DMARC)
Stalwart 관리자 UI의 `Settings > Domains` 메뉴에서 제공하는 레코드를 DNS에 등록하여 신뢰할 수 있는 발신자임을 증명해야 합니다.
