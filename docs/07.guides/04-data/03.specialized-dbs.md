# Specialized Databases Guide

Usage and configuration guide for NoSQL, Graph, and Time-Series databases in `hy-home.docker`.

## 1. Document Databases (MongoDB, CouchDB)

### MongoDB
- **Replica Set**: Configured for high availability.
- **Usage**: General-purpose NoSQL for dynamic schemas.

### CouchDB
- **Sync**: Multi-master replication for mobile/edge sync.
- **Cluster**: 3-node HA configuration.

## 2. Wide-Column Stores (Cassandra)

### Cassandra
- **Scalability**: Linearly scalable distributed cluster.
- **Schema**: Performance-first data modeling (Query-driven).

## 3. Graph Databases (Neo4j)

### Neo4j
- **Cypher**: Query language for relationships.
- **Use Case**: Deep hierarchies, network analysis, recommendation engines.

## 4. Time-Series Databases (InfluxDB)

### InfluxDB
- **Flush**: Optimized for high-velocity IoT and metric data.
- **SQL/Flux**: Query languages for analytical time-series analysis.
