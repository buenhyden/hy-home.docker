# Object Storage (S3-Compatible)

> Guide for MinIO and SeaweedFS storage layers.

## Overview

`hy-home.docker` provides two S3-compatible layers:

- **MinIO**: High-performance, rich features, used for infra logs (Loki/Tempo).
- **SeaweedFS**: Distributed small-file optimization, high throughput.

## Usage Guide (MinIO)

### Initial Setup

Buckets like `tempo`, `loki`, and `cdn` are created automatically via `minio-init`.

### CDN Hosting

Upload assets to the `cdn` bucket. They are served via the public Traefik route if configured correctly.
