# Specialized Databases

> Guide for MongoDB, Cassandra, InfluxDB, and Vector engines.

## Overview

Specialized databases handle unique data shapes and query patterns, complementing our core SQL and Cache layers.

## Databases Covered

### 1. MongoDB (Document)

Resilient NoSQL document store with a 3-node replica set. Ideal for JSON-like data and flexible schemas.

### 2. InfluxDB (Time-Series)

Optimized for performance metrics and analytical data tracking.

### 3. Qdrant (Vector)

Powering the RAG (Retrieval-Augmented Generation) layer for AI agents.

### 4. Cassandra (Columnar)

High-throughput columnar storage for large-scale distributed datasets.

- [Infrastructure Source](../../../infra/04-data/cassandra/README.md)

### 5. Neo4j (Graph)

Relationship-first graph database for complex connectivity analysis.
