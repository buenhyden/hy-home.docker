# Integrated Stacks (Supabase)

> Comprehensive guide for managing the self-hosted Supabase ecosystem.

## Overview

The `supabase` stack provides an all-in-one backend-as-a-service (BaaS) layer, integrating PostgreSQL, authentication, and RESTful APIs.

## Service Components

- **GoTrue**: Authentication and user management.
- **PostgREST**: Automated RESTful API for Postgres.
- **Realtime**: WebSocket-based database change broadcasting.
- **Kong**: Central API Gateway for Supabase services.

## Management Tasks

### Accessing Supabase Studio

The Studio UI is accessible at `https://supabase.${DEFAULT_URL}` once deployed. Use the configured admin credentials found in your secrets.
