# Standards

프로젝트 운영에 필요한 규칙과 기준 문서를 한 곳에 모읍니다.

## 참고 문서

- `AGENTS.md` (에이전트 규칙)
- `ARCHITECTURE.md` (아키텍처 개요)
- `CONTRIBUTING.md` (기여 규칙)
- `.markdownlint.json` (문서 규칙)

## Infra 문서 규칙

- 각 `infra/<번호-카테고리>/<서비스명>/README.md`에는 **File Map** 섹션을 포함합니다.
- 서비스 실행 방식(Core/Optional/Standalone/Placeholder)을 `infra/README.md` 기준으로 유지합니다.
- 명령어 예시는 실제 실행 가능한 값으로 작성합니다(placeholder 금지).
