## What is an ADR?

ADR은 소프트웨어 설계에 있어 중요한 의사 결정을 문서화하는 기술입니다. 이는 향후 프로젝트 유지보수 시 "왜 이렇게 설계되었는가?"에 대한 맥락을 제공합니다.

[ADR Template Guide](./0001-record-architecture-decisions.md)를 참조하여 새로운 결정을 작성하십시오.
