# 🎓 hy-home.docker Learning Hub

Welcome to the **hy-home.docker** educational center. This project is designed not just as a functional home infrastructure, but as an **"Infrastructure-as-Theory"** laboratory.

## 🏛️ Learning Framework

The knowledge here is organized into four distinct tiers, bridging the gap between academic theory and production implementation.

### [Tier 1: Computer Science (Theory)](../../.agent/skills/self-learning-guide/references/)

Foundational logic, data structures, and distributed systems protocols.

- **Key Topics**: CAP Theorem, Raft Consensus, HNSW Graph Search, OIDC Protocols.

### [Tier 2: Computer Engineering (Hardware/OS)](../../.agent/skills/self-learning-guide/references/02-os-virtualization/theory.md)

The interaction between software and hardware/kernel.

- **Key Topics**: Linux Namespaces/Cgroups, CPU Scheduling, SHM (Shared Memory) Optimizations.

### [Tier 3: Software Engineering (Practice)](../../.agent/skills/self-learning-guide/references/05-observability/theory.md)

Professional practices for maintaining scalable systems.

- **Key Topics**: GitOps Lifecycle, SRE Golden Signals, Secret Management.

### [Tier 4: Hands-on Mini-Projects](./roadmap-v1.md#tier-4-hands-on-mini-projects-implementation)

Practical implementation tasks to verify your understanding.

- **Key Projects**: Custom Controller/Operator, Distributed Lock Service, Vector RAG Pipeline.
